## 🏗 安装

```bash
# npm install
$ npm install @lins-material/components --save

# yarn install
$ yarn add @lins-material/components

# pnpm install
$ pnpm i @lins-material/components
```

## 🔨 使用

```ts
import { Battery } from '@lins-material/components';

export default () => {
  return (
    <Battery />
  )
}
```

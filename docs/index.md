## 🏗 安装

```bash
# npm install
$ npm install sensoro-design --save

# yarn install
$ yarn add sensoro-design

# pnpm install
$ pnpm i sensoro-design
```

## 🔨 使用

```ts
import { Battery } from 'sensoro-design';

export default () => {
  return (
    <Battery />
  )
}
```
